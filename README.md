# Supervised ML: Top 5 Models

This repository was created as the companion codebase for my supervised machine learning course: https://www.udemy.com/course/supervisedml
It is designed to help beginners move from concept-level understanding to actually running and modifying ML code.

This repository contains beginner-friendly notebook examples for five foundational supervised machine learning models: linear regression, logistic regression, k-nearest neighbors, decision trees, and random forests. Each notebook follows the same basic machine learning workflow: prepare the data, split into training and testing sets, train the model, evaluate performance, and interpret the results.

## Models Covered

| Notebook | Model | Problem Type | Main Idea |
|---|---|---|---|
| `01_linear_regression.ipynb` | Linear Regression | Regression | Predicts a numeric value by learning a line of best fit |
| `02_logistic_regression.ipynb` | Logistic Regression | Classification | Predicts a class using probability |
| `03_k_nearest_neighbors.ipynb` | k-Nearest Neighbors | Classification | Predicts based on similar nearby examples |
| `04_decision_tree.ipynb` | Decision Tree | Classification | Learns feature-based decision rules |
| `05_random_forest.ipynb` | Random Forest | Classification | Combines many decision trees for more stable predictions |

## Project Goal

The goal of this project is to make supervised machine learning easier to understand through clear, runnable examples. Instead of only explaining the theory, each notebook shows how the model works in real Python code.

## This Repository Demonstrates...

- Supervised machine learning workflows
- Train/test splitting
- Model training with scikit-learn
- Regression and classification tasks
- Accuracy, precision, recall, confusion matrices, and mean squared error
- Feature scaling for distance-based and linear classification models
- Comparing simple models with more flexible models
