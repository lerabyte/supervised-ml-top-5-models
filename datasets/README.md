## Included Dataset

- `episode_1_2_used_car_dataset.xlsx` — a small example dataset used for regression practice.

## Dataset Structure

The dataset is organized in a spreadsheet format, where each row represents one example and each column represents a feature or target value.

In general:

- **Rows** = individual data examples
- **Feature columns** = input variables used by the model
- **Target column** = the value the model is trying to predict

For example, in a used car dataset, the features might describe the car, while the target column could represent the car’s price.

## Note

This dataset is included for educational and demonstration purposes. It is meant to help users practice loading data, training models, and evaluating predictions in Python.
